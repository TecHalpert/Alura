Get - Trás resultado, mostra os dados na url.
O protocolo HTTP define que o GET deve ser utilizado apenas para acessar os dados, mas HTTP, como protocolo, não pode impedir o desenvolvedor de fazer algo diferente. 

Post - Envia resultado e não mostra os dados na url.
